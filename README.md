# Chat-Enabled Online Shopping Store

This project is a chat-enabled online shopping store developed as part of Database Systems and Computer Networks coursework. The system integrates core e-commerce functionality with real-time customer communication.

## Project Overview
The application allows users to browse products, place orders, and communicate with sellers or support staff through a chat feature. The project emphasizes database integration and basic networking concepts for efficient client-server interaction.

## Key Features
- Product listing and management
- User registration and order processing
- Real-time chat functionality
- Database-driven data storage and retrieval

## Technologies & Concepts
- Database Management Systems
- Computer Networks (client-server communication)
- SQL queries and schema design
- Basic networking and system integration

## Project Type
Academic / University Project
